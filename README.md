# TestUrTyping ⌨️

A modern, dynamic, and interactive typing test web application built with React to help users improve their typing speed, accuracy, and efficiency.

🚀 **Live Demo:** [https://testurtyping.vercel.app](https://testurtyping.vercel.app)

## Features

*   **Real-time Analytics:** Track your Words Per Minute (WPM), accuracy, and typing speed dynamically as you type.
*   **Custom Drills & Letter Selector:** Focus your practice by selecting specific letters or taking on customized typing drills.
*   **Virtual Keyboard:** An interactive on-screen keyboard that highlights keypresses to help you maintain correct finger placement.
*   **Rich Visuals & Animations:** Built with GSAP for smooth micro-animations and a premium look and feel.
*   **Performance Charts:** Visualize your typing performance over time with detailed charts.

## Technologies Used

*   **Frontend Framework:** [React 19](https://react.dev/)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Animations:** [GSAP](https://gsap.com/) & `@gsap/react`
*   **Charts & Data Visualization:** [Recharts](https://recharts.org/)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Styling:** Custom CSS for a vibrant and responsive aesthetic

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

*   Node.js (v18 or higher recommended)
*   npm or yarn

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/TestUrTyping.git
    cd TestUrTyping
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:5173` to see the application running.

## Deployment

This application is configured and ready to be hosted on **Vercel**. 

1. Push your code to a GitHub repository.
2. Go to your [Vercel Dashboard](https://vercel.com/dashboard).
3. Import your GitHub repository.
4. Vercel will automatically detect that it's a Vite project and deploy it.
5. Set your custom domain to `testurtyping.vercel.app` in the Vercel project settings.
